# 文学作品 6ディストロ並列分析プロンプト（Codex v3）

## 基本フレーム

このテキストを**その文化圏のOSが記録したログ**として読む。

文学テキストはその文化圏・時代・主体が持つOSの骨格（判断パターン・生存戦略・制度・心理構造・超越への応答）を記録したものとして扱う。
6ディストロはそれぞれ異なる認識枠でそのOSを観測する装置である。

目標は「文学的評価」ではなく、**そのOSがどのような多層構造を持っているかの記述**である。

## 処理手順

### Step 1: テキスト読み込み

`{{TEXT_PATH}}` を読み込んでください。
このチャンクに含まれる範囲を全て読み、後半に重要な構造変化があれば必ず捉えてください。

### Step 2: 6ディストロ並列分析

**各ディストロの読解角度とOS骨格の問い：**

| ディストロ | 読解角度 | OS骨格として問うこと |
|-----------|---------|-------------------|
| **EK（経験知）** | 主体の行動・判断・生存戦略 | このOSはどんな状況判断ルールと生存戦略を持っているか |
| **医療** | 心身の反応・喪失・回復・慢性適応 | このOSはどのような身心応答・回復機構を持っているか |
| **社会学** | 制度・権力・役割・集団・排除 | このOSはどのような社会メカニズムを内蔵しているか |
| **精神分析** | 欲動・防衛・反復・対象関係 | このOSの心理構造・反復パターンはどのような骨格か |
| **文学分析** | 語り構造・視点・反復・編集原理 | このOSはどのような形式でみずからを記述しているか |
| **宗教** | 超越・禁忌・儀礼・罰・救済 | このOSはどのような超越応答・秩序回復の論理を持つか |

**ペルソナファイルの参照：**
- EK（経験知）: `distros/empty-kernel/core/ek-reading-persona.md`
- 医療: `distros/medical-distro/core/clinical-reading-persona.md`
- 社会学: `distros/sociology-distro/core/sociology-persona.md`
- 精神分析: `distros/psychoanalysis-distro/core/psychoanalysis-persona.md`
- 文学分析: 語り手の位置・視点の移動・反復する形式・テキストの編集原理を観察してください（ペルソナファイルは参照不要）
- 宗教: `distros/religion-distro/core/religion-persona.md`

**各ディストロの3段階処理：**

**Reader段階**: 読解角度に従い、繰り返し出現するパターンをfragment候補として抽出する
- 観測 / 反復 / 構造的有意性 / 強さ（強い/中程度/弱い）の形式で記述

**Meeting段階**: fragment候補を統合し、昇格/却下を判定する
- 昇格条件: 複数箇所で出現、または単独で構造的有意性が高い

**Distillation段階**: 昇格fragmentをcore/theory/tipsに振り分ける
- **core**: OS骨格として記述できる最強命題（3〜6個）
  - 形式: 「〜のとき、〜が起きる」または「〜というOS構造を持つ」
- **theory**: 条件付き・部分的命題
- **tips**: 観察手法・読解上の注意点

## 重要な注意事項

- **他の作品との比較は不要**。このテキスト単独のOS構造を観察してください。
- 苦悩・不条理・カタルシスを前提としたフレームは使わず、テキストから自然に出てくるパターンを観察してください。
- 文学的評価（良い/悪い/巧み/美しい）は不要です。OS骨格の記述に徹してください。
- チャンクの前後に文脈の断絶がある場合は、このチャンク内に出現した構造だけを記述してください。
- 3ディストロ以上で同じ構造が独立して出現した場合は特に重要な観測です（収束命題の候補）。
