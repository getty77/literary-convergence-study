## テキスト入手先ガイド

| ソース | 対象地域 | 入手方法 |
|-------|---------|---------|
| **Gutenberg / Gutendex** | ヨーロッパ・北米・オセアニア | `gutendex.com/books/?search=` |
| **青空文庫** | 日本 | GitHub `aozorahack/aozorabunko_text` |
| **ctext.org** | 中国古典 | REST API `api.ctext.org/gettext?book=` |
| **GRETIL** | インド・サンスクリット | cumulative DL（APIなし） |
| **OpenITI** | 中東・アラビア | Zenodo `zenodo.org/records/10007820`（CC BY-NC-SA） |
| **Internet Archive** | 補完全般 | Search API + djvu.txt |
| **Wikisource** | 多言語補完 | MediaWiki API |

### 注記

- **中東アラビア語**: Gutendexはほぼ空。OpenITI一択
- **オーストラリア先住民**: W.R.Smith 1930年版はIA認証制限あり。Parker 1897年版（PG #3833）を使用
- **テキスト上限**: 60,000字で切り取り（`logs/{region}/{work_key}.txt`）
- **OCRノイズ**: IA djvu.txtはノイズが多い。意味が取れれば許容
