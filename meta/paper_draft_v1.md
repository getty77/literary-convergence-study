# 10古典文学テキストから観測される行動骨格とその収束構造

---

## 0. Abstract

本研究は、紀元前2100年頃から19世紀までの10文化圏の古典文学テキストを対象に、
6視点の並列分析によって各文化圏の思考・行動の骨格構造を抽出する探索的記述研究である。

対象は著作権切れの公開テキスト10作品（メソポタミア・ギリシャ・中世ヨーロッパ・
ロシア・日本・中国・インド・アラビア・マヤ・オセアニア）とし、
LLM学習データからの補完は一切行っていない。

テキストを行動・身体・社会・心理・語り・超越の6視点で独立に分析し、
545のコア命題を記録した。このうち、3文化圏以上で独立に同一構造が出現した
収束命題を40件特定した。

文化的接触が考えにくい組み合わせ（例：ギリシャ×マヤ×メソポタミア）において、
「行動停止→外部権威による再起動」「役割逸脱の連鎖崩壊」「身体が認知に先行する」
等の構造が6文化圏で独立出現した。

本研究は探索的記述研究であり、収束命題の普遍性を主張しない。
手法・コーパス・全出力データは以下のリポジトリで公開する。
https://github.com/getty77/literary-convergence-study

---

## 1. 研究の問い

「文学テキストから、その文化圏の思考・行動の骨格構造を観測できるか」

文学テキストは、その文化圏・時代・主体が蓄積してきた判断・行動・制度・心理の記録として読める。
複数の独立した読解視点から同一テキストを分析し、反復する構造を骨格命題として抽出することで、
そのテキストが記録している行動文法を観測できると仮定した。

---

## 2. 手法

### 2.1 コーパス選定

以下の3条件を全て満たすテキストのみを対象とした。

1. **著作権切れ（PD）**: 公開テキストとして取得可能なこと
2. **実テキスト**: LLM学習データからの知識補完を一切行わないこと
3. **文化的正典性**: その文化圏で継承・引用され続けてきた正典であること

| 作品 | 文化圏 | 年代 | 文字数 | テキスト入手先 |
|-----|--------|------|--------|--------------|
| ギルガメッシュ叙事詩 | メソポタミア | 紀元前2100頃 | 246k | Project Gutenberg #11000 |
| オデュッセイア | ギリシャ | 紀元前800頃 | 679k | Project Gutenberg（Butler訳） |
| 神曲 地獄篇 | 中世ヨーロッパ | 1308〜1320 | 681k | Project Gutenberg（Longfellow訳） |
| 罪と罰 | ロシア | 1866 | 1,135k | Project Gutenberg（Garnett訳） |
| 源氏物語 | 日本 | 1008頃 | 537k | 青空文庫（与謝野晶子訳） |
| 論語 | 中国 | 紀元前400頃 | 196k | ctext.org（中国語原文） |
| バガヴァッド・ギーター | インド | 紀元前200頃 | 144k | Project Gutenberg（Besant訳） |
| 千夜一夜物語 | アラビア | 8〜14世紀 | 1,001k | Project Gutenberg（Lane訳） |
| ポポル・ヴフ | マヤ | 1550頃記録 | 95k | Project Gutenberg #56550（Goetz訳） |
| ポリネシア神話集 | オセアニア | 1855記録 | 529k | Internet Archive（George Grey） |

計10作品、総文字数約5,242k字。
翻訳テキストを使用した作品については、翻訳者の解釈が混入する可能性がある（→ 5節）。

### 2.2 分析視点

6視点はそれぞれ独立した観測装置として機能する。各視点は「そのテキストが記録している骨格構造の何を観測するか」を明確にした上で独立して処理される。視点間の調整は行わない。

| 視点 | 観測対象 | 骨格として問うこと |
|-----|---------|-----------------|
| 行動 | 主体の判断・行動・生存戦略 | どんな状況判断ルールと生存戦略を持っているか |
| 身体 | 心身の反応・喪失・回復・慢性適応 | どのような身心応答・回復機構を持っているか |
| 社会 | 制度・権力・役割・集団・排除 | どのような社会メカニズムを内蔵しているか |
| 心理 | 欲動・防衛・反復・対象関係 | 心理構造・反復パターンはどのような骨格か |
| 語り | 語り構造・視点・反復・編集原理 | どのような形式で自己を記述しているか |
| 超越 | 超越・禁忌・儀礼・罰・救済 | どのような超越応答・秩序回復の論理を持つか |

各視点の分析は以下の3段階で処理した。

1. **抽出段階**: 繰り返し出現するパターンを断片候補として列挙する
2. **判定段階**: 断片候補を統合し、昇格（複数箇所出現または構造的有意性が高い）か却下かを判定する
3. **定式化段階**: 昇格断片をコア命題（core）・補助観察（theory）・読解注記（tips）に振り分け、コア命題は「〜のとき、〜が起きる」または「〜という骨格構造を持つ」の形式で定式化する

### 2.3 分析パイプライン

全文テキストを200k字単位でチャンク分割し、各チャンクをLLMで6視点並列分析した後、
チャンク別の結果を別のLLMで統合・重複除去してコア命題を確定した。
確定した命題は全作品分をJSONでフラット化し、横断分析に供した。

実装の詳細（スクリプト・プロンプト・モデル指定）はリポジトリを参照。
https://github.com/getty77/literary-convergence-study

### 2.4 収束の定義

本研究では「収束」を2段階で定義する。

**作品内収束**: 同一テキストにおいて、3視点以上が独立に同一の骨格構造を捕捉した命題。
テキスト内での構造的有意性が高いことを示す。

**文化圏横断収束**: 異なる文化圏の3作品以上において、独立に同一の骨格構造が出現した命題。
文化的接触が考えにくい組み合わせ（例: ギリシャ×マヤ×メソポタミア）での出現を優先的に記録した。

収束は「普遍性の証明」ではなく「複数の独立した観測装置が同構造を捕捉した」という事実の記録である。
収束命題の解釈は読者に委ねる。

## 3. 作品別観測（A節）

各作品から観測された行動骨格の代表命題を示す。
命題は6視点から独立して抽出し、複数視点で同構造が出現したものを優先して掲載した。

### ギルガメッシュ叙事詩（メソポタミア、紀元前2100頃）

観測数: 47命題（6視点）

**代表命題**
- **行動**: 死の可能性が明示されるとき、撤退よりも名を立てる行動が選ばれる
- **身体**: 死や致死的変化が意識化されるとき、静止よりも治癒探索と移動が起きる
- **社会**: 危険な企図が出るとき、共同体はまず制止し、次に条件付きで支援へ移る
- **心理**: 自己に酷似した他者が現れるとき、融合ではなく格闘が先行し、その後に対象安定が起きる
- **語り**: 出来事そのものより、どの断片からどう復元したかという編集過程を前景化して自己を記述する
- **超越**: 不死を求めるとき、「神々が生を保持し、死を人間に課した」という超越秩序が呼び出される

- **語り**: テキストは最初に外部視点で客を扱い、歌と質問を経て、最後に一人称の本格回想へ接続する

## 4. 横断観測（B節）

6視点・10作品から独立して出現した収束構造を記録する。
収束命題は全40件特定した（詳細: `section4_convergence_raw.md`）。
以下には、3作品以上かつ文化的接触が考えにくい組み合わせ（ギリシャ×マヤ×メソポタミア等）で確認された36件を掲載する。
出現文化圏の記載がない命題は `section4_convergence_raw.md` を参照のこと。
解釈は行わず、構造の記述に留める。

---

### 行動視点の収束命題（5命題）

**B-行動-1**: 直接突破の封鎖 → 迂回・偽装・情報先行による局面反転
定式: 「正面からの力が通じない状況で、主体は力比べを回避し、相手の情報構造を先に把握してから非対称な手段で反転させる」
出現: ポポル・ヴフ / 千夜一夜 / オデュッセイア / ポリネシア神話 / ギルガメッシュ

**B-行動-2**: 行動停止 → 外部権威・上位秩序の注入による再起動
定式: 「自力判断が機能停止するとき、主体は上位に位置づけられた他者・秩序・説明を受け取ることで行動が再開する」
出現: 神曲 / バガヴァッド・ギーター / 罪と罰 / ギルガメッシュ / ポリネシア神話 / 論語

**B-行動-3**: 喪失対象の代替物連鎖による持続
定式: 「回復不能な喪失が生じたとき、主体は喪失を処理せず、似姿・後継者・別対象への連想的置換を繰り返して機能を維持する」
出現: ギルガメッシュ / 源氏物語 / 罪と罰 / 千夜一夜 / ポリネシア神話

**B-行動-4**: 危機における単一証拠による信頼確定
定式: 「主体の同一性や能力が疑われるとき、累積的説得ではなく一回の圧倒的な実演・身体的証拠によって信頼関係が一挙に確定する」
出現: オデュッセイア / 千夜一夜 / ギルガメッシュ / ポリネシア神話 / 源氏物語

**B-行動-5**: 核心情報の条件付き封鎖と解放
定式: 「主体は核心情報を封鎖し続け、聴き手が自分を傷つけられない状況に入ったと判断したとき初めて開示する」
出現: 神曲 / 罪と罰 / 千夜一夜 / オデュッセイア / ポポル・ヴフ

---

### 身体視点の収束命題（8命題）

**B-身体-1**: 重大局面では認知より先に身体反応が起きる
定式: 「重大な危機・決断・衝突が起きるとき、思考・判断より先に震え・失神・口渇・発熱感が起きる」
出現: バガヴァッド・ギーター / 罪と罰 / 千夜一夜 / ギルガメッシュ / 源氏物語 / ポリネシア神話

**B-身体-2**: 洗浄→保温→休息の回復シーケンス
定式: 「極度の損傷・汚染・漂流のあと、洗浄→保温/栄養→休息の順次処理が回復の主経路になる」

## 5. 限界と記録

### 5.1 翻訳テキストのバイアス

10作品中8作品は翻訳テキストを使用した（英訳またはProject Gutenberg公開版）。
翻訳テキストには翻訳者の解釈・時代的語彙・文化的枠組みが混入する。

具体的な問題として：

- **語彙選択の解釈混入**: 原語の多義語が翻訳時に単一解釈へ収束する（例：アラビア語の「アダブ」を「礼節」と訳す場合、身体的・道徳的・審美的含意が失われる）
- **構造の透明化**: 翻訳は原文の曖昧性・反復・語順を整理する傾向があり、行動・判断の骨格として機能していた形式的構造が見えにくくなる
- **翻訳時代のバイアス**: ギルガメッシュ（Andrew George訳）・ポポル・ヴフ（Dennis Tedlock訳）はいずれも20世紀の学術翻訳であり、その時代の文化人類学的枠組みが解釈に影響している

英訳を介さなかった2作品においても同様の問題がある。源氏物語（与謝野晶子訳）は原典の平安古文から現代日本語への変換を経ており、論語（中国語原文）においても現代的解説・注釈への参照が分析に混入した可能性がある。

本研究では翻訳テキストの使用を「制限として記録した上で許容する」立場をとる。
探索的段階では原語分析の厳密さより、複数文化圏への到達範囲を優先した。

### 5.2 正典選別バイアス

本研究で使用した「文化的正典」は、現在まで継承・引用・制度的保存を経たテキストである。
この選別プロセス自体が、観測される骨格構造に方向性を与えている可能性がある。

- **生存者バイアス**: 現存するテキストは、権力・宗教機構・教育制度によって保存されたものに限られる。同時代の民衆的記録・異端的テキスト・口承文化のより多くの部分は残存していない
- **正典化の均質化**: 繰り返し引用・解釈される過程で、正典テキストはその文化圏の「理想形」として整形される。実際の行動文法より規範的行動文法が記録されている可能性がある
- **文字文化への偏り**: パブリックドメインの文字テキストを使用したことで、文字文化を持たない・持っていた時期が短い文化圏（ポリネシア・マヤ等）は19世紀以降の記録集成を経由した。この採録・編集過程に西洋研究者の枠組みが入っている

これらのバイアスは「観測された行動骨格構造が文化圏の実態を反映しているか」という問いへの答えを留保させる。本研究が観測しているのは、「その文化圏が正典として継承することを選んだテキストの中に現れる構造」である。

### 5.3 N=10の探索的規模

本研究のコーパスはN=10の探索的パイロット規模である。
収束命題の解釈においては以下の点に注意が必要である：

- **統計的検証不可**: N=10では命題の出現頻度に統計的有意性を付与できない。「6文化圏で出現」は記述的事実であり、確率論的意味における「有意」ではない
- **文化圏の代表性**: 各文化圏から1作品のみを抽出しており、その作品が文化圏全体の行動文法を代表するとは言えない。例えば「ギリシャ古典」はオデュッセイア単体で観測しており、ソクラテスの対話篇・悲劇文学とのズレは検証していない
- **作品内の収束と文化圏間の収束の混同**: 本研究では作品ごとにコア命題を記録し、その後文化圏間で照合した。しかし作品内の反復が強い命題は、その文化圏の重みを過大に評価させる構造になっている

本研究の結果は「収束候補の記述と列挙」として扱い、「普遍命題の確定」として扱わない。

### 5.4 LLM分析装置のバイアス

本研究の分析エンジンはLLM（OpenAI Codex / Anthropic Claude Sonnet）である。
テキストから行動骨格構造を抽出する装置自体が、大規模な人間テキストデータで訓練されている点は根本的な限界である。

- **訓練データへの汚染リスク**: LLMは学習データとして本研究対象テキストの多くを含んでいる可能性が高い。「テキストを読んで構造を抽出する」という指示に対し、実際のテキスト分析ではなく訓練時に形成された解釈パターンを返す可能性がある
- **プロンプト感度**: 6視点分析プロンプトの語彙選択（「行動骨格」「思考・行動の構造」等）が、抽出される命題の方向性を規定している。異なるプロンプトで異なる命題群が出現する可能性は排除できない
- **収束命題の循環リスク**: LLMが複数文化圏のテキストに対して類似の構造を返す場合、それは文化横断的収束の証拠である可能性と、LLMが訓練データから同一パターンを一貫して投影している可能性の両方がある。この区別は現時点では困難である
- **再現性の未確認**: 同一テキスト・同一プロンプトを再実行した場合に同一命題群が出現するかは確認していない。LLMの出力確率的性質により、完全再現は保証されない

これらの限界を踏まえ、本研究は「LLMという分析装置を通じて観測された構造の記述」として位置づける。観測装置の限界を越えた主張は行わない。

## 6. 今後の研究

### 6.1 コーパス拡張と収束命題の検証

本研究はN=10のパイロット段階であり、収束命題の統計的検証には至っていない。
次の段階として以下を想定する。

- **コーパス拡張（N=50〜100）**: 同一文化圏から複数作品を追加し、1作品代表性のリスクを下げる。特に現状で1作品しか含まれていないオセアニア・マヤ・インドの補強が優先課題である
- **反証事例の積極的探索**: 収束命題の反証例（同構造が出現しない文化圏・作品）を同一手法で記録し、収束の輪郭を描く
- **原語テキストへの移行**: 可能な範囲で翻訳テキストから原語テキストへ切り替え、翻訳バイアスの影響を測定する比較実験を行う

### 6.2 個別著者・個人ログへの適用

本手法は単一テキストに対しても適用可能である。

- **著者の時系列分析**: 単一著者の全著作を時系列順に処理し、作品間で骨格構造がどう変化するかを観測する（例：ドストエフスキーの前期・後期作品間の変化）
- **個人記録への適用**: 日記・書簡・SNSログ等の個人的テキストに同手法を適用することで、文化圏テキストと個人テキストの骨格構造を同一フォーマットで比較する経路が開ける。ただし倫理的配慮とプライバシー管理が前提となる

### 6.3 分析手法の精緻化

- **再現性の検証**: 同一テキスト・同一プロンプトを異なるモデル・異なるタイミングで複数回実行し、命題の再現率を測定する
- **プロンプト感度の測定**: 分析プロンプトの語彙・構造を系統的に変化させ、出力命題への影響を記録する
- **他手法との接続**: トピックモデリング・共起分析・定量テキスト分析等の既存手法との比較により、LLM抽出命題の独自性と限界を明確化する

### 6.4 収束命題の意味解釈

本研究は収束命題の列挙にとどまり、その解釈は読者に委ねた。
後続研究として以下の問いが残る。

- 収束命題は認知的普遍性（人間の共通認知構造）を反映しているか、それとも文化伝播・共通の生態学的制約の産物か
- 収束命題のうち、時代・地域を問わず現代人の行動においても観測されるものはどれか
- 収束しなかった命題（文化圏固有の骨格構造）が持つ意味は何か
