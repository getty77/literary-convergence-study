# 10古典文学テキストから観測される行動骨格とその収束構造

---

## 0. Abstract

本研究は、10文化圏の古典文学テキストを6視点で並列分析し、
各文化圏に観測される思考・行動の骨格構造（以下、行動骨格）を抽出する探索的記述研究である。

- **コーパス**: 著作権切れ（PD）テキスト10作品、紀元前2100頃〜20世紀初頭
- **分析視点**: 行動・身体・社会・心理・語り・超越の6視点
- **結果**: 511コア観測命題を記録
- **位置づけ**: 本研究は探索的記述研究である。観測された構造の解釈は読者に委ねる。

---

## 1. 研究の問い

「文学テキストから、その文化圏の思考・行動の骨格構造を観測できるか」

文学テキストは、その文化圏・時代・主体が蓄積してきた判断・行動・制度・心理の記録として読める。
複数の独立した読解視点から同一テキストを分析し、反復する構造を骨格命題として抽出することで、
そのテキストが記録している行動文法を観測できると仮定した。

---

## 2. 手法

### 2.1 コーパス

著作権切れのPDテキストのみを対象とした。LLM学習データからの補完は一切行っていない。

| 作品 | 文化圏 | 年代 | 文字数（原文） |
|-----|--------|------|--------------|
| ギルガメッシュ叙事詩 | メソポタミア | 紀元前2100頃 | 251k |
| オデュッセイア | ギリシャ | 紀元前800頃 | — |
| 神曲 地獄篇 | 中世ヨーロッパ | 1308〜1320 | — |
| 罪と罰 | ロシア | 1866 | 1,130k |
| 源氏物語 | 日本 | 1008頃 | — |
| 論語 | 中国 | 紀元前400頃 | — |
| バガヴァッド・ギーター | インド | 紀元前200頃 | — |
| 千夜一夜物語 | アラビア | 8〜14世紀 | 1,010k |
| ポポル・ヴフ | マヤ | 1550頃 | 96k |
| ポリネシア神話集 | オセアニア | 1855記録 | 529k |

### 2.2 分析視点

6視点はそれぞれ独立した観測装置として機能する。

| 視点 | 観測対象 |
|-----|---------|
| 行動 | 主体の判断・行動・生存戦略 |
| 身体 | 心身の反応・喪失・回復 |
| 社会 | 制度・権力・役割・排除 |
| 心理 | 欲動・防衛・反復パターン |
| 語り | 語り構造・視点・編集原理 |
| 超越 | 超越・禁忌・儀礼・救済 |

### 2.3 分析パイプライン

```
全文テキスト取得（PDテキストのみ）
    ↓
チャンク分割（200k字/チャンク、20k字オーバーラップ）
    ↓
LLMによる6視点並列分析（チャンク単位）
    ↓
チャンク結果の統合・重複除去（LLM）
    ↓
コア命題（C-n）の抽出・記録
```

### 2.4 収束の定義

3視点以上で独立に同一構造が出現した命題を「収束命題」として記録した。
収束は「普遍性の証明」ではなく「複数の観測装置が同構造を独立に捕捉した」という事実の記録である。

## 3. 作品別観測（A節）

各作品から観測された行動骨格の代表命題を示す。
命題は6視点から独立して抽出し、複数視点で同構造が出現したものを優先して掲載した。

### ギルガメッシュ叙事詩（メソポタミア、紀元前2100頃）

観測数: 47命題（6視点）

**代表命題**
- **行動**: 死の可能性が明示されるとき、撤退よりも名を立てる行動が選ばれる
- **身体**: 死や致死的変化が意識化されるとき、静止よりも治癒探索と移動が起きる
- **社会**: 危険な企図が出るとき、共同体はまず制止し、次に条件付きで支援へ移る
- **心理**: 自己に酷似した他者が現れるとき、融合ではなく格闘が先行し、その後に対象安定が起きる
- **語り**: 出来事そのものより、どの断片からどう復元したかという編集過程を前景化して自己を記述する
- **超越**: 不死を求めるとき、「神々が生を保持し、死を人間に課した」という超越秩序が呼び出される

---

### オデュッセイア（ギリシャ、紀元前800頃）

観測数: 72命題（6視点）

**代表命題**
- **行動**: 他者の保護意志と忠誠がまだ確定していないとき、主体は必要最小限だけ語り、歓待・護送・忠誠確認が整ってから深部の身元情報を出す
- **身体**: 海難・漂流・長期緊張で心身が限界に達するとき、主体は長い睡眠に落ち込み、そこから再び行動可能域へ戻る
- **社会**: 見知らぬ来訪者が受け入れられるとき、歓待は家内的な施しに留まらず、集会・船団・贈与へ拡張されて制度化される
- **心理**: 論争的な空白があるとき、主体は大胆で一貫した総体仮説を立て、その内部整合性で不安を封じる
- **語り**: テキストは最初に外部視点で客を扱い、歌と質問を経て、最後に一人称の本格回想へ接続する
- **超越**: 主体がどれほど望んでも、帰還は神々が道を開くときにのみ次段階へ進む

### 神曲 地獄篇（中世ヨーロッパ、1308〜1320）

観測数: 48命題（6視点）

**代表命題**
- **行動**: 進路を見失い自力回復が失敗すると、主体は権威ある先達をガイドとして採用する
- **身体**: 正道喪失のとき、恐怖・息切れ・疲労・震えが一体化して出現する
- **社会**: 個人の危機が語られるとき、同時に都市史・党派・階級編成が展開される
- **心理**: 内的混乱が生じると、暗い森・獣・荒野として外在化される
- **語り**: 作品は旅を語る前に、旅を読む資格と地図を先に配布する
- **超越**: 正しい道を失うと、それは空間的な迷子ではなく霊的秩序からの離脱として現れる

- **社会**: 債務関係に入ると、人は対等な主体ではなく追跡・監視・回収の対象になる
- **心理**: 主体は対象を憎悪しつつ、その対象に繰り返し接近する
- **語り**: 三人称叙述は主体の思考に密着するが、時折外部から補助線を引いて読者の判断を保留させる
- **超越**: 苦しみは単なる不運ではなく、罰であり浄化でもある可能性として保持される

- **社会**: 寵愛が既存の序列を乱すとき、集団は陰湿な排除で均衡を回復しようとする
- **心理**: 原対象を喪失したとき、主体は似姿を追い、同型の対象で欠如を埋めようとする
- **語り**: 帝の宮廷史から源氏の夜の逸脱へと視野を狭めることで、制度の内面化を記述する
- **超越**: 愛・出生・喪失・再会は、前世の縁や業として意味づけられる

- **社会**: 父子・君臣・長幼の役割が守られるとき、社会秩序が再生産される
- **心理**: 欲望が生じるとき、礼と義が即時充足を遅延させ、昇華へ向かわせる

### バガヴァッド・ギーター（インド、紀元前200頃）

観測数: 30命題（6視点）

**代表命題**
- **行動**: 近親・情動・役割衝突で行動停止するとき、主体は自力判断を停止し、より上位の秩序説明を受け取ることで再起動する
- **身体**: 親族殺害と義務遂行が衝突するとき、震え・口渇・脱力・発熱感・失神感として身心が同時に破綻する
- **社会**: 私的情愛と公的役務が衝突するとき、社会秩序は個人感情より役割遂行を優先させる
- **心理**: 愛着対象を自分の手で失う想像が立ち上がるとき、主体は行為能力を失い退行する
- **語り**: 局所的な戦場の逡巡は、対話の反復によって宇宙論・倫理・実践論を含む総合体系へ展開される
- **超越**: 人間的な善悪判断が停止するとき、主体は超越的法へ帰依し、その命令によって行為を正当化する

### 千夜一夜物語（アラビア、8〜14世紀）

観測数: 67命題（6視点）

**代表命題**
- **行動**: 近しい他者の裏切りを目撃したとき、個別事例は世界一般の規則へ拡張され、極端な予防行動が起きる
- **身体**: 裏切りや脅威に曝されたとき、蒼白・不眠・食欲低下・失神・沈鬱が起き、身体が物語の一次記録媒体になる
- **社会**: 君主の私的損傷が公的命令に変換されると、婚姻制度と家族再生産が国家単位で破壊される
- **心理**: 愛着対象の裏切りに遭うと、主体は個別対象を失い、対象カテゴリ全体を危険として分裂的に把握する
- **語り**: 一つの危機が語られると、その内部に別の判例譚が挿入され、処刑・終結が迫るたびに新たな語り手が別の逸話を差し出して語りの停止を回避する
- **超越**: 主体が誓いを立て神名を持ち出すとき、行為は超越的監視のもとに置かれ、誓約がある場合にのみ一時的な秩序回復が可能になる

- **語り**: 一度で完成せず、生成と失敗と再記述を重ねながら世界像が組み立てられる
- **超越**: 被造物が神々への想起と奉仕を果たせないとき、破壊と再創造が起きる

- **心理**: 宇宙的分離が起きるとき、光と世界が生まれると同時に、兄弟間の報復と持続的敵対も生じる

- **超越**: 宇宙秩序は無機的法則ではなく、神的親族の分離・怒り・報復によって成立する

## 4. 横断観測（B節）

6視点・10作品から独立して出現した収束構造を記録する。
以下は、3作品以上かつ文化的接触が考えにくい組み合わせ（ギリシャ×マヤ×メソポタミア等）で確認された命題を優先して掲載した。
解釈は行わず、構造の記述に留める。

*詳細データ: `section4_convergence_raw.md`*

### 行動視点の収束命題（5命題）

**B-行動-1**: 直接突破の封鎖 → 迂回・偽装・情報先行による局面反転
定式: 「正面からの力が通じない状況で、主体は力比べを回避し、相手の情報構造を先に把握してから非対称な手段で反転させる」
出現: ポポル・ヴフ / 千夜一夜 / オデュッセイア / ポリネシア神話 / ギルガメッシュ

**B-行動-2**: 行動停止 → 外部権威・上位秩序の注入による再起動
定式: 「自力判断が機能停止するとき、主体は上位に位置づけられた他者・秩序・説明を受け取ることで行動が再開する」
出現: 神曲 / バガヴァッド・ギーター / 罪と罰 / ギルガメッシュ / ポリネシア神話 / 論語

**B-行動-3**: 喪失対象の代替物連鎖による持続
定式: 「回復不能な喪失が生じたとき、主体は喪失を処理せず、似姿・後継者・別対象への連想的置換を繰り返して機能を維持する」
出現: ギルガメッシュ / 源氏物語 / 罪と罰 / 千夜一夜 / ポリネシア神話

**B-行動-4**: 危機における単一証拠による信頼確定
定式: 「主体の同一性や能力が疑われるとき、累積的説得ではなく一回の圧倒的な実演・身体的証拠によって信頼関係が一挙に確定する」
出現: オデュッセイア / 千夜一夜 / ギルガメッシュ / ポリネシア神話 / 源氏物語

**B-行動-5**: 核心情報の条件付き封鎖と解放
定式: 「主体は核心情報を封鎖し続け、聴き手が自分を傷つけられない状況に入ったと判断したとき初めて開示する」
出現: 神曲 / 罪と罰 / 千夜一夜 / オデュッセイア / ポポル・ヴフ

### 身体視点の収束命題（8命題）

**B-身体-1**: 重大局面では認知より先に身体反応が起きる
定式: 「重大な危機・決断・衝突が起きるとき、思考・判断より先に震え・失神・口渇・発熱感が起きる」
出現: バガヴァッド・ギーター / 罪と罰 / 千夜一夜 / ギルガメッシュ / 源氏物語 / ポリネシア神話

**B-身体-2**: 洗浄→保温→休息の回復シーケンス
定式: 「極度の損傷・汚染・漂流のあと、洗浄→保温/栄養→休息の順次処理が回復の主経路になる」
出現: オデュッセイア / ギルガメッシュ / ポリネシア神話 / 罪と罰 / 神曲

**B-身体-3**: 外部ガイドがいると苦痛は通過可能な負荷に変わる
定式: 「自己調整で抜け出せない状態において、外部の支持者が介入するとき、苦痛は通過計画が与えられると耐えうる負荷に変わる」
出現: 神曲 / 罪と罰 / ギルガメッシュ / ポリネシア神話 / バガヴァッド・ギーター

**B-身体-4**: 喪失の長期化は身体症状として沈殿し周囲にも波及する
定式: 「喪失・帰還不明が長期化するとき、睡眠障害・反芻・涙・身体衰耗が複数人物にまたがって持続する」
出現: オデュッセイア / 源氏物語 / 千夜一夜 / ポリネシア神話 / 罪と罰

**B-身体-5**: 怒り・執着の持続が回復回路を閉鎖する
**B-身体-6**: 悲嘆の儀礼通過が秩序回復の必須回路になる（論語 / オデュッセイア / 源氏物語 / ギルガメッシュ / ポリネシア神話）
**B-身体-7**: 限界に達した身体は長い睡眠で行動可能域へ戻る
**B-身体-8**: 身体崩壊の多様性・個体差構造

### 語り視点の収束命題（5命題）

**B-語り-1**: 遅延正当化構造（告白・頂点・到達の遅延装置）
定式: 「本来の中心事象が存在するとき、テキストはそれを直接提示せず、迂回・試験・挿話の連鎖によって到達を遅延させ、その遅延行為が正当化の実体となる」
出現: オデュッセイア / 千夜一夜 / 罪と罰 / 神曲 / バガヴァッド・ギーター / 源氏物語

**B-語り-2**: 同一命題の多層再記述構造（変奏による意味場の厚化）
定式: 「一つの命題が反復されるとき、それは同一内容の強調ではなく、文脈・比喩・階層を変えながら再記述されることで意味場を厚くし、単一解釈への収束を妨げる」
出現: 論語 / バガヴァッド・ギーター / 千夜一夜 / 罪と罰 / ポポル・ヴフ / ポリネシア神話

**B-語り-3**: 物質的細部の転換点占有構造（モノが物語の蝶番になる）
定式: 「物語の転換点において、抽象的説明ではなく特定の物質的細部が配置され、その物体が意味の切り替え装置として機能する」
出現: オデュッセイア / 罪と罰 / 源氏物語 / ギルガメッシュ / ポリネシア神話

**B-語り-4**: 外枠証言による個人経験の公共化構造
定式: 「個人の内的経験・告白が存在するとき、外部から観察・記録する別の声を設けることで、個人的出来事を公的に有効な記録へ変換する」
出現: バガヴァッド・ギーター / 千夜一夜 / オデュッセイア / ポポル・ヴフ / 論語

**B-語り-5**: 系譜接続による語りの継続構造（終わりを次の始まりへ変換する）
定式: 「一つの物語単位が終結に向かうとき、テキストは系譜・血統・継承の記述によって次の主体を立ち上げ、物語を止めない」

## 5. 限界と記録

- **翻訳テキスト**: 英訳テキストは翻訳者の解釈が混入する。原語テキスト（源氏物語・論語）も現代語訳のバイアスがある
- **正典選別バイアス**: 現存する「正典」は権力・制度による選別を経ている
- **N=10の探索的規模**: 本研究は探索的分析であり、命題の統計的検証は今後の課題
- **LLM抽出の限界**: 分析装置自体がLLMであり、訓練データのバイアスが構造抽出に影響する可能性がある
